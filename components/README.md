# Componentes do manual

Os componentes visuais do manual são implementados com classes CSS reutilizáveis em `style.css` e padrões semânticos nos fragmentos HTML:

- `.callout`, `.tip`, `.important`, `.warning`, `.critical`
- `.checklist`
- `.timeline` e `.timeline-item`
- `.steps` e `.step`
- `.faq-item`
- `.card`, `.badge`, `.table-wrap`, `.screenshot`, `.diagram`

Este diretório fica reservado para evoluções futuras de componentes HTML compartilhados sem introduzir frameworks.
